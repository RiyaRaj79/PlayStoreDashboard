# 📱 Google Play Store Dashboard

An interactive data visualization dashboard built using Python and Plotly.

## 📊 Charts Included
- Chart 1: Grouped Bar Chart → 3 PM to 5 PM IST
- Chart 2: Choropleth Map → 6 PM to 8 PM IST
- Chart 3: Dual Axis Chart → 1 PM to 2 PM IST
- Chart 4: Time Series Line Chart → 6 PM to 9 PM IST
- Chart 5: Bubble Chart → 5 PM to 7 PM IST
- Chart 6: Stacked Area Chart → 4 PM to 6 PM IST

## 🛠️ Built With
- Python
- Pandas
- Plotly
- Streamlit
- Pytz

## 📂 Dataset
Google Play Store Apps dataset from Kaggle

# PlayStoreDashboard
Interactive Google Play Store Dashboard with 6 Plotly charts including Grouped Bar, Choropleth Map, Dual-Axis, Time Series, Bubble, and Stacked Area charts. Each chart is time-restricted to specific IST windows. Built with Python, Pandas, Plotly and Streamlit.
